# FlightMath
Calculate some db information

## Инструкция по запуску

1. Восставновите базу из папки DB
2. Указать актуальную строку подключения к базе данных
3. Выполните миграции EF Core
4. Запустите проект

## Описание API

1. /Statistics - возвращает общую статистику по рейсам
Включает такие поля как:
* общий вес груза
* дистанция
* выручка пропорционально дистанции
* количество накладных
* и др.

2. /SegmentsInfo - возвращает информацию по рейсам с раздеением на строки по отрезкам пути ( A-B,B-C разделено на A-B и B-C)

3. /FlightRate?origin=XXX&destination=YYY - вычисляет средний трафик на маршруте ( выручка / вес груза)
* Если выруска = 0, рейс исключается из суммирования
* Результат - среднее арефметическое выручка/вес всех полетов по данному рейсу
* Если не найдено - вернется 0
